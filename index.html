Integrations

VNaaS integrates with your favorite IDEs and brings auto variable naming goodness to your workflow. Just press Ctrl + F13, and you shall have a variable name that rocks. The IDE plugins auto-detect the current language, and give you a variable name with correct case -- see, we solved one more problem!

Of course, you want it as a module, don't you? We're working on shipping an npm module super soon, so you'd be just able to import VNaaS, and get done with it.
